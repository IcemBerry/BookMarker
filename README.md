## BookMarker
Book Marker是一个用于管理阅读进度以及笔记的网站, 主体使用Spring与Mybatis进行搭建

## 构建框架
    Spring
    MyBatis
    BootStrap
    ECharts
    BootStrap-table
    MySQL
    Alibaba Druid
    Apache Maven
    Apache Log4j
